# Black-Jack

## Repository
https://github.com/ixl220004/Black-Jack.git

## Description
This game will let the user play Jackblack.

## Features
- Feature 1
	- will draw from a deck from a list and shuffle once the deck is low.
- Feature 2
	- will check if the user or the dealer goes over 21
- Feature 3 
	- will count the ammount of chips the user has and allow the user to bet more chips
