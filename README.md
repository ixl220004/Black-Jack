This game is a the classic BLACKJACK game that most gamblers know. It's my favorite so 
I decided to create a verion i could play. If you know how to play blackjack it's pretty straight forward:
1. Place your bets.
2. You and the dealer get 2 cards. you can either draw cards or stand.
3. Try to get the highest hand value without going over 21.
4. the dealer has to draw until he hits atleast 17. if he goes over you win.

I made multiple verions of the game, Blackjack_V1 is the basic game played in the terminal, it uses a text file as
it's deck. Blackjack_Game_V2 is another version that imports V1 and adds betting and multiple round play. I accidentliy 
made these two verions thinking it would be easy to impliment them into pygames (it was not). I basically had to restart
my project within pygames, but atleast i was able to learn from that. V1 and V2 arne't used in the final game but if you 
get the chance to try them out go ahead.

The final game plays in a pygames window and is super neat. The video explains most of the features, but one I forgot to 
mention is that the game is going through a deck throughout the whole game. when it hits less then 10 cards it reshuffles.
You don't see it on screen and it only happens behind the scenes, but if you wanted a realistic game (or learn how to count cards)
then now you have it. While i finished most of the technical stuff and gamplay, I'll admit the game could use more visual 
flair. in the future, if i get the chance to keep working on this, I would add a casino mode,  my game only uses one deck, but
casinos use multiple decks mixed together to make it harder to count cards. I'd also would've loved to add more visuals like
a casino background with a dealer, or mabye flashier text for when you win or lose.

In the end, I'm proud of my work. I showed the V2 to a classmate and he got invested in it for a good minute. showed the final version 
to my mom and she loved it. so I think I've accomplished my goals here.
